"Me"

Exploratory Hypertext Visualization Project

This interactive radial tidy tree represents the different paths of a male or female based on end-user choices which ultimately leads to if path steered individual into an anxiety condition. Utilizing the repurposed tidy tree as an abstract to create a non-linear narrative and have end-user interact and apply their own choices or real world attributes in comparison to other paths. This data has been derived from societal "norms", statistics and collaborative suggestions to be utilized as sample data for this project. It is not meant to be acted on or with intent to offend.
